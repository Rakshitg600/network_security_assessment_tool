## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/network-security-tool.git
cd network-security-tool
```

2. Install required packages
```bash
pip install python-nmap
```

3. Run the application
```bash
python network_security_tool.py
```

## Usage
1. Launch the application
2. Enter target IP address in the input field
3. Click "Start Scan" to begin assessment
4. View real-time results in the dashboard
5. Generate detailed reports using the "Generate Report" button
